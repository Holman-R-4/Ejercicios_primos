2. Lógica y Código de MATLAB

Para determinar si un número $n$ es primo y obtener sus predecesores, utilizamos la función isprime y el operador de rango :.

% --- Script de Verificación de Números ---

n = input('Introduce el número: ');

% Verificamos si es primo
if isprime(n)
    fprintf('El número %d es PRIMO\n', n);
else
    fprintf('El número %d NO es primo\n', n);
end

% Generamos la cuenta regresiva
if n > 1
    anteriores = (n-1):-1:1;
    fprintf('Números anteriores: ');
    disp(anteriores);
end
